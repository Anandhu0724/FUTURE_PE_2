## 🧠 Deep-Dive: The Underlying Prompt System

The app’s intelligence relies on a structured, multi-tier prompt matrix running inside Google AI Studio. Below are the precise structural instructions executing behind the user interface:

### ⚙️ Prompt 1: Direct-Response Ad Structural Core
This instruction controls the core generator logic, enforcing a rigid script timeline that balances visual cues, native screen-text overlays, and fluid conversational delivery.

```markdown
[SYSTEM ROLE]: Master Direct-Response UGC Creative Director.
[OBJECTIVE]: Convert raw brand parameters into a 45-60s unpolished vertical video script.
[CRITICAL PARAMS]:
1. Structure: Visual Hook (0-3s) ➡️ The Vent (3-15s) ➡️ The Discovery (15-40s) ➡️ Micro-CTA (40-60s).
2. Diction: Use casual human colloquialisms ("literally", "honestly", "game-changer"). Eliminate formal marketing syntax.
3. Deliverables: Explicitly split all script lines into [Visual Cue], [Text Overlay], and [Audio Line].
